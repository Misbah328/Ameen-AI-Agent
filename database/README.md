# Database

This folder contains database schemas, migrations, seed data, and database documentation for Ameen Secretary.

Main database responsibilities:

- Users
- Organizations
- Meetings
- Transcripts
- AI summaries
- Decisions
- Action items
- Tasks
- Reports
- Boards and committees
- Governance records

Recommended tools:

- PostgreSQL
- Prisma ORM
- pgvector for AI memory and semantic search
