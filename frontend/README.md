# Frontend

This folder contains the Next.js frontend application for Ameen Secretary.

Main modules:

- Authentication
- Dashboard
- Record Meeting
- Meeting Schedule
- Transcripts & Minutes
- Last Meeting
- Task & Decision Tracker
- Ask Ameen
- Report Generator
- Team Members
- Settings

Technology:

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
