# Backend

This folder will contain the backend services for Ameen Secretary.

Backend responsibilities:

- Authentication
- User management
- Organization management
- Meeting management
- Transcript management
- AI processing APIs
- Task management
- Decision management
- Report generation
- Notifications
- External integrations

Recommended backend stack:

- Node.js / NestJS
- PostgreSQL
- Prisma ORM
- REST APIs
- JWT Authentication
