# API

This folder contains API specifications, endpoint definitions, contracts, and integration documentation for Ameen Secretary.

API responsibilities:

- Authentication APIs
- User APIs
- Meeting APIs
- Transcript APIs
- AI Processing APIs
- Task APIs
- Decision APIs
- Report APIs
- Dashboard APIs
- Integration APIs

Recommended standards:

- REST API
- JSON
- OpenAPI 3.0
- JWT Authentication
- Versioning with /api/v1
