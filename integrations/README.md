# Integrations

This folder contains integration plans and future implementation files for external platforms used by Ameen Secretary.

Planned integrations:

- Zoom
- Google Meet
- Microsoft Teams
- Google Calendar
- Outlook Calendar
- Gmail
- Microsoft Outlook
- WhatsApp Business
- Slack
- Microsoft 365
- Google Drive
- OneDrive
- SharePoint

Integration goals:

- Import meeting recordings
- Sync calendars
- Send invitations
- Send reminders
- Connect documents
- Send notifications
- Support enterprise workflows
